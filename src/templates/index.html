Step 9 Fire SPARQL DESCRIBE

Start typing to copy a parsed server response to a <div> element. The click events now use the URI returned by the autocompletion to fire a SPARQL DESCRIBE query, e.g:

DESCRIBE <http://linkedlifedata.com/resource/umls/id/C0022885>

We will use the autocomplete service of the LinkedLifeData.com website. And fire the SPARQL query to the endpoint at http://linkedlifedata.com/sparql. However, the problem is that not all SPARQL endpoints properly handle JSONP/CORS so we wrap it in a server side function {{ url_for('sparql_rdf') }} (that will come in handy later anyway)
